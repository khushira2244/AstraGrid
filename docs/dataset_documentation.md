# AstraGrid Dataset Documentation

## Project

**AstraGrid / SIFT Sentinel** is a Protocol SIFT extension for autonomous cyber-physical incident response.

The demo case investigates whether a water utility anomaly is:

- an isolated water anomaly
- a direct water cyberattack
- a coordinated multi-sector attack
- a power-to-water cascade
- unsupported by available evidence
